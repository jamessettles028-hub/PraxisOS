## 🚀 Running PraxisOS for the First Time

### 1. Open PraxisOS  
You can start PraxisOS by clicking its icon on your desktop or in the Start menu. If you do not see an icon, search for “PraxisOS” in the Start menu search bar.

### 2. Allow System Permissions  
The program may ask for permission to access your files or network. Approve these requests, as they help PraxisOS manage tasks and sync with your Git tools.

### 3. Set Up Your Workspace  
PraxisOS uses Git to track your work. You do not need to know Git commands. The program handles this behind the scenes. You will see options to create or open projects.

### 4. Create Your First Project  
Click **New Project** and give it a name. PraxisOS will create a folder on your computer to store the project files.

### 5. Start Managing Tasks  
Use the interface to create tasks. PraxisOS will save changes automatically. Your tasks are stored locally and tracked using Git, so you can revert changes if needed.
