# Support

## Getting Help

### Bug Reports

Found a bug? Please [open a GitHub issue](https://github.com/Reddidgy/PraxisOS/issues/new) with:

- A clear description of the problem
- Steps to reproduce
- Your browser name and version
- Screenshots if applicable

### Feature Requests

Have a feature idea? Two ways to share it:

- **In-App Feedback**  -  Use the built-in feedback button inside PraxisOS to submit feature requests directly from the app.
- **GitHub Issues**  -  [Open an issue](https://github.com/Reddidgy/PraxisOS/issues/new) and tag it as a feature request.

### General Questions

For general questions about PraxisOS:

- Check the [README](README.md) for usage instructions and supported browsers.
- Browse [existing issues](https://github.com/Reddidgy/PraxisOS/issues)  -  your question may already be answered.

## What This Project Is Not

PraxisOS is not a SaaS product with a support desk. It is a locally-run tool with community-driven support via GitHub. We read and respond to every issue, but response times may vary.
